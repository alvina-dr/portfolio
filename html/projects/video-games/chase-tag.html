Chase Tag (online)

2022-23 Unity Photon Unity Networking 2 Socket.io Phaser 3

In october 2022 I started making a chase tag game after work. I had wanted to learn about online multiplayer for a long time and suddendly felt like it was possible with my current knowledge. It worked pretty well pretty quickly so I started talking with a friend and we created some rules to make it funny.

But with other projects in the way during my vocationnal degree, the project got left behind. The idea never left though. In April 2023, after thinking about it for so long I decided to switch it directly to web instead of using Unity and then building to WebGL. The old solution was too long to test and there were huge delay between the testers.

I had already used Phaser 3 before with Chloé au Musée so it was the logical way to go. For the client server communication I used Socket.io. The website is deployed on Heroku (that we had already used before during my DHE with Ruby On Rails).

The project is still in progress today and if you want to have a try, there's a link on the right to the old and new version of the chase tag.
