#Online multiplayer

At the start of october 2022 I started making an online chase tag game after work. I had wanted to learn about multiplayer for a long time and suddendly felt like it was possible with my current knowledge. It worked pretty well pretty quickly so I started talking with a friend and we created some rules to make it funny. This game is still a work in progress but you can try a prototype by clicking on the button on the right !
