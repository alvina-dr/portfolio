Pokemon Memory

#React

In November 2021, we had a React workshop during my DHE. As I already had learned a bit of JS thanks to freeCodeCamp, I decided to realise a pokemon memory with my partner.

The website is in english and is responsive. There is a first tab to explain the game and then the game itself. The goal is to discover as much pairs as you can in 60 seconds. The pairs of pokemon that you find will go into your pokedex. The aim ? To catch them all !

I am really proud of this funny game and consider this workshop a success.
